Cuestionario 1: BD vs DBMS

1. ¿Qué es una base de datos - BD?

Es la coleción de diferentes archivos en los que aparecen multiples ocurrencias de un mismo tipo de registro, interrelacionados que contienen información que no se repite y que abarque las necesidades de la empresa.[1]

2. ¿En qué casos es conveniente usar una base de datos?

Siempre que se trabaja con un gran volumen de datos, y que la informacion necesaria se maneja en varios archivos y esta informacion esta interrelacionada; como puede ser el caso de una empresa con informacion de inventario, usuarios, ventas, etc.[1]

3. ¿Que es un sistema de gestión de bases de datos - BDMS?

Básicamente se trata de un conjunto de programas para acceder a una coleccion de datos que estan relacionados entre sí. Su principal objetivo es dar un ambiente amigable para la consulta y administracion de los datos. Esto significa no solo la gestión de los mismos datos, sino medidas de seguridad para protejerlos.[1]

4. Enlista y explica las funciones/responsabilidades que tienen los DBMS

  • Crear y organizar la base de datos
  • Establecer y mantener las trayectorias de acceso de los datos para asegurar un acceso rápido y confiable a ellos
  • Manejar los datos de los usuarios con concurrencia
  • Mantener en todo momento la integridad y la seguridad de los datos del cliente
  • Registrar el uso de la base de datos[2][3]
  • Administración de Diccionario de datos : almacenamieto de definiciones de elementos y sus relaciones con un diccionario
  • Administración en almacenamiento de Datos: crean y administran estructuras de datos para su almacenamiento, y guardan la base de datos en archivos fisicos para su acceso.
  • Adminstracion de seguridad: privilegios que se le asignan a cada rol, pedendiendo de la funcion que ejercen.
  • Administracion de recuperacion y respaldos: procediminetos que actuan en caso de alguno de estos eventos, asegunrando la recuperacion de los datos
  • Administracion de integridad de datos: promueve las reglas de integridad maximizando la consistencia
  • Interfaces de conunicacion de BD: dan interfaces entre bases de datos, como por ejemplo dentro de un navegador web. [4]

5. Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos

Como sólo he trabajado en proyectos pequeños en la escuela, actualmente no creo tener algun proyecto el cual hubiera sido mas sencillo con la implementacion de una base de datos.
