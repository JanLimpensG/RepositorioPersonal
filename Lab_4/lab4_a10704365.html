¿Qué diferecias y semejanzas hay entre JavaScript y Java?

La principal diferencia entre los dos, es que Java es un lenguaje de programación, meintras que JavaScript, no. Sin embago ambos comparten la misma sintaxis. En JS no existe la declaración de tipos de variables a usar, ya que se encarga de ello automáticamente. Además, generamlente, JS no puede escribir en disco duro, por lo que se le considera un lenguaje seguro en Internet. Otra diferencia, es que Java necesita un IDE y una máquina virtual, mientras que JS sónlo necesita un procesador de texto para operar. Y finalmente, Java es un lenguaje que esta orientado a objetos, y JavaScript esta basado en prototipos.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5)

  • getDate() : te regresa el dia del mes (entre 1 y 31)
  • getDay() : te regresa el dia de la semana (de lunes a domingo)
  • getFullYear() : regresa el año con sus 4 dígitos
  • getSeconds() : devuelve los segundos entre 0 y 59
  • setTime() : actualiza la fecha completa

¿Qué métodos tienen los arreglos? (Menciona al menos 5)

  • concat() : une a dos o mas arreglos
  • indexOf() : busca un elemento dentro del arreglo y regresa su indice
  • join() : junta elementos de un array en una cadena con separador
  • revrese() : invierte el orden de los elementos
  • split() : convierte un string en un array

¿Cómo se declara una variable con alcance local dentro de una función?

LAs variables locales en JavaScript se declaran con la palabra reservada 'let'. Este tipo de variables "mueren" cuando se llega al final de las llaves dentro de las cuales fueron creadas, haciendolas una mejor opción que al usar 'var' (variables globales).

¿ Qué implicaciones tiene usar variables globales dentro de funciones?

El manejo de dichas variables se vuelve mucho mas complejo al tener cuidar el uso de la misma variable fuera de un 'scope' local, en caso de que la quieras usar dentro de una funcion. Si se llega a cambiar el valor de ella en alguna otrra funcion, su valor cambiara a lo largo todo el programa.

¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué pordías usar esto en una aplicación web?

El metodo split, ya que puedes convertir un String a un array, y comparar caracter por caracter. Esto lo puedes usar en una aplicación web para tomar instrucciones o inputs como strings y usarlos de esta manera internamente en tu programa.
