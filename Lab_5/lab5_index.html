¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?

Para poder detener errores antes de que estos sean escritos dentro de los datos del servidor. Ya que esto funciona como un primer filtro de la informacion que vamos a guardar.
